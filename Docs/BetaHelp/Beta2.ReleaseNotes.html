Welcome to Track Maker 2.0 Beta Release (Priscilla Beta2: version 2.0.614.20359)

Welcome to Track Maker 2.0. This release of the Track Maker includes many improvements, new features, and bug fixes.

DISCLAIMER: This version of the Track Maker is incomplete and comes as-is with no guarantee of quality, and has bugs and missing functionalities that would not exist on a final release. Do not use this for production purposes, only for testing and evaluation purposes.

New features include:

  • Layers

  • Pan and zoom

  • ATCF and HURDAT2 import (only ATCF import working as of right now)

  • HURDAT2 importing

  • A new and vastly improved project format

  • Integrated the ACE calculator and advisory generator!

  • Node time indicator (coming soon: season start time indicator)

  • Improved menu structure

  • Many bug fixes and reliability improvements

  • Logging

  • The ability to define custom storm types

  • More preinstalled basins and category systems

  • Previews!

Guide

Just extract the binaries and open Track Maker.exe to start! Please make sure to test all functionality!

Issues fixed since Beta1:
  • The Save Current Project button now defaults to TProj2 format.
  • Previews and the "Season Start Time" display are now implemented!
  • Some "duplicate layer" bugs.
  • Fixed the Import/Export button being cut off on some import/export windows.
  • Improved ATCF export functionality, the coordinates output make somewhat more sense.
Other Changes Since Beta1:
  • The "Open" button is back to "Import" again, I think it reads better considering its child menu options.
  • Removed most beta branding
Known Issues (don't report these):
  • Using the zoom controls while panning will reset the screen position to the top-left corner (working on this).
  • You can't enable and disable a layer if it's not the selected layer.
  • The Help button doesn't work. This is because there is no help yet.
  • HURDAT2-format importing is not currently implemented.
  • Some of the text is cut off.
Q & A
Q: The Track Maker crashed or displayed an unexpected error or fatal error message!

A: In the directory you opened the Track Maker, there should be a "Priscilla-Log-*.txt" file created each time you launch the Track Maker.
Find the most recent one and either send it to me with a description of the issue or file as an issue on GitHub (at https://github.com/Cosmo224/TrackMaker) WITH THE LOG FILE ATTACHED! Otherwise I will not be able to debug it.

Q: I want to request a feature!

A: DM me on Discord (Starfrost#9088) or file an issue on GitHub and I will consider it.

Q: I found a bug not listed above!

A: DM me on Discord with details of the bug and your most recent Priscilla-Log-*.txt file; alternatively, file an issue on GitHub WITH THE MOST RECENT LOG ATTACHED!

Q: I want to launch the ACE Calculator or Advisory Generator individually without launching the Track Maker first!

A: You can. Run the Track Maker with the command-line argument 
-initacecalc
to start the ACE Calculator,
and 
-initadvgen
to start the Advisory Generator.

There will be shortcuts for this created by the installer in the final version.

Q: What does Priscilla mean? I see it everywhere!

A: It was the codename for Track Maker 2.0, back when I wasn't sure if it was version 1.5 or 2.0. If you're interested, the next version after 2.0 is codenamed Iris.

If you have any other questions, please send me a DM on Discord (Starfrost#9088). Have fun! Your feedback is vital to the success of this project!
